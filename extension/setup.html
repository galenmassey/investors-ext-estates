🐛 Troubleshooting

Native Helper Not Connecting?

  1. Check estate_helper.log in the download folder
  2. Verify registry entry: reg query "HKCU\Software\Google\Chrome\NativeMessagingHosts\com.investors.estate_helper"
  3. Ensure extension ID matches in manifest
  4. Restart Chrome completely

PDFs Not Downloading?

The portal may require authentication for PDFs. Check the documents folder for _metadata.txt files with URLs for manual download.
